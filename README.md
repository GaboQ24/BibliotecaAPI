# BibliotecaAPI

API REST desarrollada con ASP.NET Core y SQL Server para la gestión de una biblioteca.

## Tecnologías utilizadas

* C#
* ASP.NET Core
* Entity Framework Core
* SQL Server
* Swagger/OpenAPI

## Funcionalidades

* Gestión de libros
* Gestión de autores
* Gestión de usuarios
* Gestión de préstamos
* Gestión de géneros
* Migraciones de base de datos
* Manejo centralizado de errores mediante middleware personalizado

## Arquitectura

El proyecto utiliza una arquitectura basada en controladores y Entity Framework Core para el acceso a datos, implementando operaciones CRUD mediante endpoints REST.
